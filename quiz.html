Part D — Philippines Challenge

True or False — Decide if each statement is correct.  判斷下列敘述是否正確。

5 pts
D1 — True / False
A launch angle of 45° gives the maximum flight distance for an air rocket on flat ground.
在平地上，45° 的發射角度可以達到空氣火箭最遠的飛行距離。
D2 — True / False
The gripper of a robotic arm is used to rotate the base of the arm.
機械手臂的夾爪是用來旋轉手臂底座的。
D3 — True / False
When you say "Can you show me how to assemble it?" you are asking someone to teach you a skill.
當你說「Can you show me how to assemble it?」時，你是在請別人教你一項技能。
D4 — True / False
In a science experiment, data refers to the information and numbers you collect from testing.
在科學實驗中，數據（data）是指你在測試中收集的資訊和數字。
D5 — True / False
"That's interesting! In my school, we use GIGO kits." is a good phrase for sharing with international students.
「That's interesting! In my school, we use GIGO kits.」是一個很適合與國際學生交流分享的句子。
